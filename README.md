# mini_active_record
Coding a mini active_record base gem step by step

run with:

```shell
bundle install
ruby db/seeds.rb
```

```shell
ruby my_app.rb
```
Or
```shell
rackup -p 4567
```

View at: [http://localhost:4567](http://localhost:4567)

Build local gem
```
gem build mini_active_record.gemspec && gem install ./mini_active_record-1.0.0.gem
```
